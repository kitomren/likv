# Ideer til utviklingen av likv med Python 
## likv overordnet
- skrive om det gamle C-programmet for likviditetsbudsjett i python for å lære python
- lage et web-skjema for å legge inn kostnader

- bruke XML eller lignende for å modellere kostnadene
- sjekke innebygde datofunksjoner i python og bruke dem hvis de er gode nok

## iterasjoner
- script som leser kostnader og lager en CSV-fil
- script som leser kostnader og lager CSV-fil med datoer

- sortering av CSV-fil i kronologisk rekkefølge
- Python filter som leser kostnader og regner saldo fortløpende
